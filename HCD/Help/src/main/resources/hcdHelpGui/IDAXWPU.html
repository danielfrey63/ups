Schritt 4: Taxonomische Stufe festlegen

Innerhalb einer Stoffliste können Sie genau eine taxonomische Stufe auswählen. Damit grenzen Sie Ihren Lernstoff auf die Taxa dieser taxonomischen Stufe ein.

So legen Sie die aktuelle taxonomische Stufe fest:

  1. Klicken Sie auf die Schaltfläche . Ein Menü mit den möglichen taxonomischen Stufen erscheint.

  2. Wählen Sie aus dem Menü die gewünschte taxonomische Stufe aus.

  3. Bestätigen Sie Ihre Wahl mit einem Klick (RETURN).

Die taxonomische Stufe wird in das Feld aktuelle taxonomische Stufe übernommen.

Beachten Sie, dass die Auswahl an taxonomischen Stufen im Menü von der gewählten Stoffspitze und der gewählten Stoffliste abhängt. Wenn sich die gewünschte taxonomische Stufe nicht im Menü befindet, wählen Sie eine andere Stoffspitze oder ggf. eine andere Stoffliste aus.

Beachten Sie ferner, dass die taxonomischen Stufen, die keine der Stoffspitze untergeordneten Taxa besitzen, grau dargestellt werden und nicht anwählbar sind.
