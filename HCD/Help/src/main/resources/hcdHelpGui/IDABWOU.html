Aktivieren des Modus Lehrgang und Abfrage

So aktivieren Sie den Modus Lehrgang und Abfrage:

  1. Öffnen Sie das Dialogfenster Assistent zur Modus-Auswahl.

  2. Selektieren Sie das Register Lernen.

  3. Wählen Sie Lehrgang und Abfrage.

  4. Klicken Sie doppelt auf den gewählten Eintrag, oder klicken Sie einmal auf OK (Alt+U).

Der Modus Lehrgang und Abfrage wird im Hauptfenster geöffnet.
