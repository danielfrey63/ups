Steinfrucht

Eine Steinfrucht ist eine meist 1samige Schliessfrucht aus 1 Fruchtblatt. Das Exokarp ist häutig, das Mesokarp fleischig, das Endokarp jedoch ist verholzt und bildet einen Stein, der den Samen enthält.
