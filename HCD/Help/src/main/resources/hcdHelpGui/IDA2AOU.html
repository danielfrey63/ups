Starten der Herbar CD-ROM Version 2

Stellen Sie vor einem Start der Herbar CD-ROM Version 2 sicher, dass die originale CD-ROM in Ihrem Computer eingelegt ist und dem Betriebssystem zur Verfügung steht. Wenn die CD-ROM nicht eingelegt ist und die Bilder nicht gefunden werden, wird das Programm mit einer entsprechenden Fehlermeldung beendet.

Wenn Sie die Herbar CD-ROM Version 2 zum ersten Mal benutzen, erscheint der Assistent zur Modus-Auswahl.

Hier wählen Sie, mit welchem Modus die Herbar CD-ROM Version 2 beginnen soll. Wenn Sie hier den Dialog abbrechen oder das Fenster schliessen, wird die Applikation nach einer Bestätigung beendet.
