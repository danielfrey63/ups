Schritt 3: Basisliste

So legen Sie die Basisliste fest:

  1. Klicken Sie auf die Schaltfläche (Alt+L). Ein Dialogfenster mit den Stofflisten erscheint.

  2. Suchen Sie die gewünschte Stoffliste im Suchfeld oder direkt im Listenfeld.

  3. Bestätigen Sie Ihre Auswahl mit einem Klick auf die Schaltfläche OK (RETURN).

Die neue Basisliste wird als aktuelle Basisliste eingetragen.

Beachten Sie, dass die Basisliste nicht verändert werden kann, wenn es nur die Gesamtliste im System hat.
