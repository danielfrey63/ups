Schritt 3: Stoffspitze festlegen

Innerhalb einer Stoffliste haben Sie die Möglichkeit, genau eine Stoffspitze festzulegen. Sie grenzen damit Ihren Lernstoff auf die zur Stoffspitze gehörenden Taxa ein.

So legen Sie die Stoffspitze fest:

  1. Klicken Sie auf die Schaltfläche . Ein Dialogfenster erscheint, in dem alle Taxa der aktuellen Stoffliste darstellt werden, hierarchisch gegliedert in Form eines Baums.

  2. Wählen Sie aus dem Baum das gewünschte Taxon aus oder suchen Sie es mit Hilfe des Suchfeldes.

  3. Bestätigen Sie Ihre Wahl mit einem Klick auf die Schaltfläche OK (RETURN).

Das Taxon wird in das Feld aktuelle Stoffspitze übernommen.

Beachten Sie: Die Auswahl an Taxa im Dialogfenster hängt von der gewählten Stoffliste ab. Wenn sich das gewünschte Taxon nicht im Angebot an Taxa befindet, wählen Sie eine andere Stoffliste aus.
