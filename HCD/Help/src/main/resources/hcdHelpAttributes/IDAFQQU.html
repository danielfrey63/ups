Einseitswendig

Wenn Organe nur auf eine Seite abstehen, nennt man dies einseitswendig. So sind z.B. die Ährchen Dach-Trespe (Bromus tectorum, Poaceae) alle auf eine Seite geneigt, der Blütenstand ist also einseitswendig.
